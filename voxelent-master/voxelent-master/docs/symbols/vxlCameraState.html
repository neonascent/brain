Class vxlCameraState

Stores a camera state so it can be retrieved later. This is the base for view landmarks.
Defined in: CameraState.js.

Class Detail

vxlCameraState(camera, camera)
Creates a vxlCameraState object and associates it with a vxlCamera. This association is unique. each camera has one vxlCameraState object. The vxlCameraState object allows to save/retrive camera states for the camera it is associated with during the construction.
Author: Diego Cantor.
